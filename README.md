# Active SPAC
Get a list of active SPAC(s) from [spacHero.com](https://www.spachero.com/).

## Commands
* `-s` or `--symbol`:
  * Search for that SPAC and display it if it's found
* `-i` or `industry`:
  * Search for a specific industry of SPACs
* `-w` or `--write-to-file`:
  * Write the results to a file; JSON or CSV
